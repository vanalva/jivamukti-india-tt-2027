Card surfaces on u-theme-brand

accommodation_card and testimonial_card default to a white content slab (--swatch--light-100) for use on neutral sections. Inside burgundy sections (.u-theme-brand), override rules remove that fill so the card frame sits directly on brand color with readable text.

Selector Behavior Tokens
.u-theme-brand .accommodation_card Transparent background; text and amenities inherit brand-readable color. transparent, --swatch--brand-text; borders use inherited --_theme---border (≈ --swatch--brand-text-o20 on brand).
.u-theme-brand .testimonial_card Same: no white slab on testimonials over burgundy. Same as above. Quote and checkmark inherit theme.

No extra HTML classes are required: wrap the section with u-section u-theme-brand (or any container with theme variables), and the descendant rules apply automatically.

Venue logo (india_venue_logo)

u-content-wrapper is a column flex container; flex items default to align-self: stretch, so a direct-child img with a fixed height was stretching to full row width. india_venue_logo sets align-self: flex-start, object-fit: contain, and a sane max-width so the Karmabhumi mark keeps its aspect ratio.

Karmabhumi / Land of action (india_land_section)

Matches the WordPress block: full-width swiper (#landActionSwiper), then a cream panel (india_land_panel uses --swatch--secondary-100) with enlarged script logo (india_land_karmabhumi_logo), uppercase kicker, and two equal text columns (india_land_columns / india_land_text) aligned on the same baseline with justified body copy. The five-tenets diagram is not used here — it belongs under the Tenets section only.

Tuition/pricing tier cards

pricing_grid holds two columns (one on small breakpoints). Standard tier uses transparent backgrounds with a tinted burgundy border so the section stays on peach (--swatch--light-200) — no white panel. Featured tier fills with --swatch--brand-500 (no shadow).

button_primary on the featured burgundy card would match the surface and disappear; overrides give the gold fill (--swatch--secondary-500) matching .u-theme-brand .button_primary. The alternate tier uses the default light-theme primary button (burgundy fill). Links include pricing_card_cta for margin-top: auto so CTAs align at the bottom.
